Welcome to mooncake's utils!
==========================================
mooncake_utils 源于作者平时工作积累的一些库和函数，本意仅供个人使用。但由于之前缺少文档，自己使用时也造成了很多困扰，所以这次抽空重新整理出了这个项目，会逐渐把这几年工作中积累的一些有用函数包含进来，方便自己也方便想用的朋友。
本项目包含了mmap高度读取、cpython编译、hadoop-streaming、日期处理、文本处理等等方面。

.. note::
  我们建议你在 `Github <https://github.com/ericyue/mooncake_utils>`_ 上star和watch `官方项目 <https://github.com/ericyue/mooncake_utils>`_ ，这样当官方有更新时，你会立即知道。

  如果你阅读在线文档时有什么问题，你可以在github上提交issue我会尽快解答。
